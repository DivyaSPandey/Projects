Step 1: Initial Setup-----
var active = 3;
var mncircles = document.querySelectorAll(".mncircle");
var sec = document.querySelectorAll(".sec");

. active = 3 â†’ Index of the currently active circle.

. mncircles selects all small circle elements.

. sec selects all section elements.

Step 2: Set initial opacity states-------------
gsap.to(mncircles[active-1], {
    opacity: 0.5
});
gsap.to(sec[active-1], {
    opacity: 1
});

.Dim all circles initially, except the one at index active-1 (because arrays are 0-indexed).

.Fully show the corresponding .sec section.

Step 3: Click event for each circle-------
mncircles.forEach(function(val, index) {
    val.addEventListener("click", function() {
        gsap.to("#circle", {
            rotate: (3 - (index + 1)) * 10,
            ease: Expo.easeInOut,
            duration: 1
        });

        greyout();

        gsap.to(this, {
            opacity: 0.5  
        });

        gsap.to(sec[index], {
            opacity: 1
        });
    });
});

. When a circle is clicked:

. Rotate #circle based on index.

. Call greyout() to dim all circles and sections.

. Highlight the clicked circle and associated section.

Step 4: greyout() function---------
function greyout() {
    gsap.to(mncircles, {
        opacity: 0.08
    });

    gsap.to(sec, {
        opacity: 0.4
    });
}

. Dims all small circles to very low opacity (0.08).

. Fades all sections to 0.4 opacity.

. This gets reset when a specific one is clicked.

Step 5: Initial rotation animation-------
gsap.to("#circle", {
    rotate: 0,
    ease: Expo.easeInOut,
    duration: 2
});

. On page load, rotate #circle to 0 smoothly in 2 seconds.
